# > Install bellow Packages from pycharm or cmd.
	1) requests -> pip install requests
	2) behave -> pip install behave
	3) pytest-bdd -> pip install pytest-bdd
	4) EnvFile -> pip install envfile 
	5) pytest -> pip install pytest
